# 0. Maven使用目的
1. Maven 是一个广泛使用的软件项目管理和理解工具。它主要由 Apache 软件基金会维护，提供了一种简便的方式来构建和管理任何基于 Java 的项目。Maven 使用一个名为“POM”（Project Object Model）的 XML 文件来描述项目的软件依赖关系、插件、版本等信息。
2. 为了自动化管理Java项目，通过插件让项目开发更加智能
